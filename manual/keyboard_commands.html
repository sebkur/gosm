Keyboard Commands

a - set navigation mode

s - set selection mode

d - set distance mode

F4 - show/hide the sidebar on the left

F5 - show/hide the movement-controls around the map

F6 - show/hide the sidebar on the right

F7 - show/hide the statusbar

F8 - show/hide the toolbar

F9 - show/hide the menubar

F10 - access the menu

F11 - toggle fullscreen

Arrow Left - move west

Arrow Right - move east

Arrow Up - move north

Arrow Down - move south

Pg Up - zoom in

Pg Down - zoom out
